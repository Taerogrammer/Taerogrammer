## Introduce

일상생활의 불편함을 기술로 해결하는 iOS 개발자 김태형입니다.  
플랫폼의 제약을 이해하고, 사용자에게 실질적인 가치를 주는 개발을 지향합니다.  
Swift 기반의 네이티브 iOS 개발은 물론, 서버리스 백엔드, 웹 프론트엔드 등 다양한 환경에 빠르게 적응하며 실전 경험을 쌓았습니다.
